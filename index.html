Aprendizagens

Tags de Títulos

As tags de título são uma das tags mais usadas no desenvolvimento web, pois ela destaca um texto comum tornando-o maior e as vezes em negrito. As tags de título possuem outra tag de fechamento, ela é composta por um "\" antes do nome da tag. Um exemplo de tag de título é o próprio título desse tópico! Existem, atualmente, 6 tipos de títulos no HTML5 usado no ano de 2023. As tags de título possuem uma sintaxae muito simples, composta por um "h" e um número de 1 a 6. Vale destacar quanto maior o número, maior será o título e mais destacado ele será. Veja um exemplo das tags de título abaixo:

  • Título 1 (h1)

  • Título 2 (h2)

  • Título 3 (h3)

  • Título 4 (h4)

  • Título 5 (h5)
  • Título 6 (h6)

Tag de Imagem

A tag de imagem, ou "img" é também é muito usada. Sua sintaxe é um pouco mais complexa que as demais, mas não muito. Sua sintaxe é composta pelo nome da tag, "img", após isso um "src" qeu é o endereço da imagem, e por fim e opcional, um "alt", ou também chamado de texto altenativo. Esse texto alternativo será visível na tela do usuário caso o endereçamento da imagem esteja incorreto, mas é aconselhado usar o alt pois deixa nosso site acessível para pessoas com deficiência!

Tag de Listas

As listas no HTML5 são divididas em dois tipos. A lista ordenada, ou seja, uma list numerada, e uma lista não ordenada, ou seja, não numerada. Vale destacar que ambas as tags das listas ordenadas e não ordenadas possuem tags de fechamento. Sua sintaxe é a seguinte:

Listas Ordenadas

As listas ordenadas, como anteriormente falada, é uma lista que é ordenada por números, seja ele números inteiros, números romanos etc. Essa lista tem o principal objetivo de criar uma lista para que o usuário possa visualizá-la em ordem. Sua sintaxe é bem simples, ela é composta por um simples "ol", ordenered list, traduzida para lista ordenada.

Lista Não-ordenadas

As listas não ordenadas, diferente das ordenadas, não são numéricas. Sua sintaxe é muito simples como a da listas ordenadas. Ela é composta por "ul", unordered list, traduzida para lista não ordenada.

Tag de Link

As tags de link são usadas para redirecionar o usuário para um site externo ou uma parte determinada do próprio site. Sua sintaxe é composta por um "a", que significa "anchor" ou "âncora" para o português, logo após um "href" qeu é o endereçamento, e opcional, um "target" que tem vários atributos que serão explicados abaixo.

Atributos

As tags de link por sua vez também possuem outro atributo além do "href", esse atributo é o "target". Esse atributo modifica o meio de redirecionamento do usúario, nesse atributo existem 4 valores. São esses valores: "_blank", "_parent", "_top", "_self".

_blank

Mesmo existindo 4 valores diferentes apenas 1 é o mais usado atualmente, ele é o "_blank". Resumindo o "_blank", ele simplismente cria uma nova aba do navegador do usuário com o link, ainda deixando o site inicial aberto.

Tag de botão

A tag de botão é também muito usada atualemente, pois é praticamente impossível encontrar um site que não possua botões. Mas os botões em si, são inúteis porque não fazem nada. Nós, desenvolvedores, precisamos adicionar algumas coisas a mais para a tag de botão realmente funcione. Para isso podemos adicionar uma tag de âncora dentro da tag de botão, adicionar eventos com a linguagem de programação Javascript etc. Sua sintaxe é simples, apenas um "button", ou botão em português.
